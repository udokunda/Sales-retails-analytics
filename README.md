# Sales-retails-analytics
End-to-end data project: SQL cleaning, EDA, ML, and Power BI dashboard.

## Project overview
Analysis of 1,000 shoe sales transactions across 7 countries, 6 brands,
and 3 sales channels. Covers data cleaning, business analysis, revenue
prediction (R²=0.9986), and interactive Power BI dashboard

## Tools used
- SQL (PostgreSQL/SQLite) — data cleaning & analysis
- Python (pandas, scikit-learn) — machine learning
- Power BI — interactive dashboard

## Key findings
- Skechers leads revenue ($269K); Nike trails at $193K
- UAE is the top market at $240K
- October & July are peak months; Aug–Sep dip 30%
- Gradient Boosting predicts revenue with R²=0.9986
